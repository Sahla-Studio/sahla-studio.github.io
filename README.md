# Sahla Studio Portfolio

Official portfolio website for **Sahla Studio**, a Moroccan product studio building simple web apps, mobile apps, tools, and systems.

## Live website

https://sahla-studio.github.io

## Stack

- Semantic HTML
- Modern CSS
- Vanilla JavaScript
- GitHub Pages

## Local preview

Open `index.html` in your browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Editing

- Page content: `index.html`
- Visual design: `styles.css`
- Menu and animations: `script.js`

Made in Morocco with purpose.
